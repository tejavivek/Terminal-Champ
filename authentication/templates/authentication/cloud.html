How to connect to our lab?


There are two ways

  1. SSH (SecureShell)
  2. PuTTY

First of all we will go through SSH as it is more easier way to get things done.
User need to download the private key we provide when the user login to their dashboard. Open any terminal (Command Prompt or Powershell(Primarily available in Windows) and move to the directory where you have downloaded the private key using cd command Now use the ssh command to prompt the login and type 'yes' when asked by the terminal BOOM! You're logged into the cloud terminal which provides linux environment Now you can follow the instructions in the blog and you are even free to explore on your own which is what we encourage for everyone to do as it keeps the enthusiasm alive and helps grow user's googling stuff skills.
